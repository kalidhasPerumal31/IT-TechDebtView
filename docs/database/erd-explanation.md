# ERD Explanation

This document explains the ERD for Tech Debt Posture & Lifecycle Manager (AG-IMP-B2-05 Phase 4).

- Posture, Risk, User, Application, and Integration are the main entities.
- Application is associated with Posture, Risk, and Integration.
- User accesses Application.
- All relationships and constraints are derived from the logical and physical schema.
