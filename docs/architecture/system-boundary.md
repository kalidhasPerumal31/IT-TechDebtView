# System Boundary

Defines the system boundary for Tech Debt Posture & Lifecycle Manager.

---

## System Scope
- Includes: Application, database, integrations (ServiceNow, endoflife.date API, Azure DevOps), user interface, access control, audit logging
- Excludes: External data sources (ServiceNow, endoflife.date API), downstream consumers (reporting, analytics)

## Actors
- End users (PMO, Leadership, Risk/Compliance, Platform/Infra, Service Owners)
- External systems (ServiceNow, endoflife.date API, Azure DevOps)
