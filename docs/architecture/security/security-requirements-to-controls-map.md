# Security Requirements to Controls Map

Maps security requirements from upstream artefacts to implemented controls.

---

| Requirement | Control |
|-------------|---------|
| RBAC | Application layer enforcement |
| SSO | Entra ID integration |
| Encryption | TLS (transit), industry-standard (rest) |
| Audit Logging | All user/system activities |
| Access Review | Quarterly, with remediation |
| Incident Response | 1-hour reporting, monthly review |
