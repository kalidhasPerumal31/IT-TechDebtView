# Security Architecture

Describes the security architecture for Tech Debt Posture & Lifecycle Manager, strictly per requirements and compliance anchors.

---

## Security Controls
- RBAC enforced at application layer
- SSO via Entra ID
- Data encryption in transit and at rest
- Audit logging for all user/system activities
- Quarterly access reviews and incident response procedures

## Compliance
- ISO/IEC 27001, NIST SSDF, OWASP ASVS 4.0, GDPR (if PII)
- All controls mapped to compliance anchors
