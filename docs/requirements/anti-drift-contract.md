# Anti-Drift Contract

This contract enforces the anti-drift rules for all requirements, ADRs, and compliance artefacts produced by AG-REQ-01-Requirements-Engineering-Agent.

---

## Anti-Drift Rules
- Downstream agents must not modify binding requirements.
- No substitution of mandated technologies without ADR + HITL.
- Requirement meaning must not be weakened or reinterpreted.
- RTM traceability must be maintained.
- Compliance obligations must not be ignored or bypassed.
- All changes to binding requirements must be escalated via ADR + HITL.
- Constraints must be propagated faithfully to all downstream artefacts.

## Enforcement
- Violation of this contract constitutes a framework breach and must be escalated immediately.
- This contract is binding for all downstream agents and human participants.
